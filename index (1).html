Resumo sobre os comados de texto no HTML

Títulos e Cabeçalhos:

h1>, h2>, h3>, h4>, h5>, h6>: Usados para criar títulos e cabeçalhos com diferentes níveis de importância
sendo h1> o mais importante e h6> o menos.

Parágrafos:

p>: Utilizado para criar parágrafos de texto.

Quebras de Linha:

br>: Cria uma quebra de linha no texto, útil quando você precisa forçar uma nova linha sem criar um novo parágrafo.
